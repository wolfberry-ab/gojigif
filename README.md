# GojiGif

**Record your screen. Get a GIF.**

A fast, private desktop app that records a region or your whole screen and exports a clean
animated GIF — with a live preview, movable selection, and pause/resume.

## Features

- **Region or whole-screen** capture → animated GIF saved to your Desktop
- **Continuous live preview** in native resolution (event-driven — updates only when the screen
  changes), with auto-recovery after screensaver / screen lock
- **Movable / resizable selection** — the region can even be moved while recording
- **Pause / resume** — paused time is excluded from the GIF timing
- **Optional burned-in timestamp** and **title card**
- **Show / hide the mouse cursor**; **adjustable FPS** (5–60, default 15)
- **In-app playback** of the last recording; **Show in Finder**
- **Privacy-first** — no telemetry, no account, no cloud. Your recordings never leave your machine

## Install

### macOS

Download the latest `GojiGif-macos-*.zip` from the
[releases page](https://github.com/wolfberry-ab/gojigif/releases/latest), unzip it, and move
**GojiGif.app** to your Applications folder.

> macOS builds are signed with our Apple Developer ID and notarized by Apple, so they launch
> without a Gatekeeper warning. On first capture, grant **Screen Recording** permission.

## Privacy & Security

GojiGif is built by [Wolfberry AB](https://wolfberry.se), a security-focused Swedish software
company.

- **No telemetry, no analytics, no account, no cloud** — GojiGif makes zero network calls
- **Your recordings and GIFs stay on your machine**, always
- The only OS permission it requests is **macOS Screen Recording**, and only when capture starts
- Report a vulnerability: **security@wolfberry.se** — see
  [security advisories](https://github.com/wolfberry-ab/gojigif/security/advisories)
